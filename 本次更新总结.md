### 2. 后台保活功能（成熟软件必备）
- **功能**：AI 输出、应用更新下载等异步任务在切换应用时不中断
- **实现**：
  - 添加 WakeLock 保持设备唤醒
  - 使用系统级后台下载服务
  - 创建统一的后台任务管理服务
  - 添加 Android 权限和配置
- **新增文件**：
  - `lib/services/background_task_service.dart`
  - `android/app/src/main/res/xml/provider_paths.xml`
- **修改文件**：
  - `lib/services/update_service.dart`
  - `lib/providers/chat_provider.dart`
  - `lib/main.dart`
  - `android/app/src/main/AndroidManifest.xml`
  - `pubspec.yaml`（添加依赖）

### 3. API 配置 404 问题
- **问题**：服务器上缺少 `backend/routers/api_config.py`
- **解决**：创建部署脚本自动上传
- **文件**：`deploy_v1.2.0.ps1`

### 4. 聊天气泡 UI 优化
- **需求**：去掉文本框，改成渐变至背景颜色
- **实现**：
  - 移除 AI 消息气泡的边框
  - 改为从 surface 颜色渐变到透明
  - 保持用户消息的渐变效果
- **文件**：`lib/features/chat/chat_page.dart`

### 5. 工具箱排序功能
- **需求**：右上角加设置，能手动拖动调组件顺序
- **实现**：

  - 创建工具排序页面
  - 使用 ReorderableListView 实现拖拽
  - 使用 SharedPreferences 保存顺序
  - 创建 Provider 管理状态
  - 支持恢复默认顺序
- **新增文件**：`lib/features/toolkit/toolkit_settings_page.dart`
- **修改文件**：

  - `lib/routes/app_router.dart`
