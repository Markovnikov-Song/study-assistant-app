# Node Quiz - 节点练习题智能生成器

## Overview

This skill generates practice questions for a specific node in a knowledge tree or mind map, using a **boundary-controlled** approach. Questions are generated based on the node's knowledge level and relationship with prerequisite/advanced nodes, ensuring exercises stay within the appropriate scope.

## Trigger Phrases (触发词)

| 触发词类型 | 示例短语 |
|-----------|---------|
| 直接命令 | "出题"、"生成练习"、"帮我出几道题" |
| 练习请求 | "练习一下这个知识点"、"刷题"、"巩固练习" |
| 复盘关联 | "类似题练习"、"再练一道"、"生成类似题" |
| 范围指定 | "根据这个知识点出题"、"围绕欧拉公式出题" |
| 英文表达 | "generate quiz", "practice questions", "give me some exercises" |

**关键触发条件：**
- 用户指向了思维导图/知识树中的某个具体节点
- 用户请求生成练习题或练习巩固
- 从复盘流程中触发的"类似题练习"步骤

## Core Concept: Boundary-Controlled Question Generation

### 设计哲学

```
传统方式（无边界）：
  出题 → 超纲内容混进来 → 用户做不出来 → 挫败感 ❌

边界控制方式：
  分析节点位置 → 确定出题范围 → 只出范围内的题 → 恰到好处 ✅
```

### 知识边界三区域

```
                    知识树
                       │
        ┌──────────────┼──────────────┐
        │              │              │
    前置知识        当前知识        后置知识
    (可出基础)     (优先出)        (暂不出)
        │              │              │
   已学过的基础    本节点核心      需要前置基础
   温故知新        重点考察        超出能力
```

### 边界规则表

| 区域 | 出题策略 | 示例（欧拉公式） |
|-----|---------|----------------|
| **前置知识** | 选少量基础题 | "什么是压杆失稳？" |
| **当前知识** | 主力出题区 | "欧拉公式的适用条件是？" |
| **后置知识** | 原则上不出 | 暂不出"柔度判别"相关题 |

## Step 1: 节点定位与分析

### 定位方式

| 方式 | 示例 | 处理逻辑 |
|-----|------|---------|
| 精确名称 | "欧拉公式" | 直接搜索匹配 |
| 层级路径 | "第十章/压杆稳定/欧拉公式" | 按路径逐级定位 |
| 错题关联 | 复盘中的错题 | 提取知识点标签 |
| 模糊搜索 | "那个临界压力的公式" | 语义匹配最近节点 |

### 节点分析维度

```python
def analyze_node(node: dict) -> dict:
    """分析节点以确定出题参数"""
    return {
        "node_id": node["id"],
        "node_name": node["text"],
        "level": node["level"],  # L1-L5
        "prerequisites": get_prerequisite_nodes(node),  # 前置节点
        "children": node.get("children", []),  # 子节点
        "knowledge_type": classify_knowledge(node),  # 概念/计算/记忆
        "difficulty_hint": node.get("difficulty"),  # 难度暗示
    }
```

## Step 4: 边界控制实现

### 边界控制伪代码

```python
def generate_questions(node: dict, count: int = 5) -> list:
    """根据节点生成练习题"""
    
    # 确定出题范围
    scope = {
        "current": [node],  # 当前节点
        "prerequisites": get_safe_prerequisites(node, max_depth=1),  # 前置节点
        "exclude": get_advanced_nodes(node),  # 排除后置节点
    }
    
    # 确定题目分布
    distribution = {
        "current": int(count * 0.7),   # 70% 当前节点
        "prerequisites": int(count * 0.3),  # 30% 前置节点
    }
    
    # 生成题目
    questions = []
    questions.extend(generate_from_nodes(scope["current"], distribution["current"]))
    questions.extend(generate_from_nodes(scope["prerequisites"], distribution["prerequisites"]))
    
    return shuffle(questions)  # 打乱顺序
```

### 边界例外情况

| 情况 | 处理方式 |
|-----|---------|
| 用户明确要求"综合练习" | 可包含后置知识，但需标注 |
| 前置知识为明显薄弱点 | 适当增加前置题比例 |
| 当前节点是基础概念 | 主要出前置基础题 |

## Notes

- **必须与知识树结构配合**：出题依赖节点的位置信息（前置/后置关系）
- **边界控制第一**：绝不主动出超出当前能力范围的题
- **题目质量优先**：宁可少出1道，也要保证题目精准
- **即时反馈**：生成的题目应附带解析，帮助即时学习
- **避免重复**：同一知识点的练习题应避免重复生成
