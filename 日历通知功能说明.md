# 日历通知功能说明

## 🎉 新增功能

已为日历事件添加系统通知功能！

### 功能特性

1. **自动通知**：创建日历事件后，系统会自动安排通知
2. **提前提醒**：默认在事件开始前 15 分钟提醒
3. **智能调度**：只为未来的事件安排通知，过去的事件不会提醒
4. **自动取消**：删除事件时自动取消对应的通知
5. **更新同步**：修改事件时间后，通知也会自动更新

### 通知内容

- **标题**：📅 事件标题
- **内容**：15 分钟后开始 · 开始时间
- **点击**：打开日历页面

### 示例

创建事件：
- 标题：数学作业
- 时间：今天 19:00
- 时长：60 分钟

系统会在 **18:45** 发送通知：
```
📅 数学作业
15 分钟后开始 · 19:00
```

## 📱 使用方法

### 1. 创建事件

1. 打开日历
2. 点击 + 按钮
3. 填写事件信息
4. 保存

✅ 系统自动安排通知！

### 2. 修改事件

1. 点击事件
2. 修改时间或其他信息
3. 保存

✅ 通知自动更新！

### 3. 删除事件

1. 点击事件
2. 点击删除按钮
3. 确认删除

✅ 通知自动取消！

## ⚙️ 权限设置

### Android 13+ 需要通知权限

首次使用时，应用会请求通知权限：
1. 点击"允许"
2. 如果拒绝了，可以在系统设置中手动开启

**手动开启方法：**
```
设置 → 应用 → 伴学 → 通知 → 允许通知
```

### 确保通知不被屏蔽

不同手机品牌可能需要额外设置：

**小米手机：**
```
设置 → 应用设置 → 应用管理 → 伴学 → 通知管理 → 允许通知
```

**华为手机：**
```
设置 → 通知 → 伴学 → 允许通知
```

**OPPO 手机：**
```
设置 → 通知与状态栏 → 通知管理 → 伴学 → 允许通知
```

**VIVO 手机：**
```
设置 → 通知与状态栏 → 应用通知 → 伴学 → 允许通知
```

## 🔧 技术实现

### 新增文件
- `lib/features/calendar/services/calendar_notification_service.dart`

### 修改文件
- `lib/features/calendar/widgets/event_form_sheet.dart`

### 通知 ID 分配
- 日历事件通知：10000 + 事件ID
- 避免与其他通知冲突

### 提醒时间
- 默认：事件开始前 15 分钟
- 可以在代码中调整 `reminderMinutesBefore` 参数

## 🐛 故障排除

### Q: 没有收到通知？

**检查清单：**
1. ✅ 通知权限是否开启
2. ✅ 事件时间是否在未来
3. ✅ 手机是否开启勿扰模式
4. ✅ 应用是否被系统杀掉（参考后台保活设置）

### Q: 通知时间不准确？

**可能原因：**
- 手机时区设置不正确
- 系统时间不准确

**解决方法：**
```
设置 → 日期和时间 → 自动设置时间
```

### Q: 如何测试通知？

**方法 1：创建即将到来的事件**
1. 创建一个 20 分钟后的事件
2. 等待 5 分钟
3. 应该收到通知

**方法 2：使用测试通知**
1. 进入：我的 → 通知设置
2. 点击"发送测试通知"
3. 立即收到测试通知

### Q: 如何关闭日历通知？

目前日历事件通知是自动的，无法单独关闭。

如果需要关闭所有通知：
```
设置 → 应用 → 伴学 → 通知 → 关闭通知
```
