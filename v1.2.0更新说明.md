
```powershell

# 清理构建
```

## ⚠️ 注意事项

### 用户设置
不同手机品牌需要用户手动设置白名单：

**小米手机：**
```
设置 → 应用管理 → 伴学 → 省电策略 → 无限制
设置 → 应用管理 → 伴学 → 自启动 → 开启
```

**华为手机：**
```
设置 → 应用启动管理 → 伴学 → 手动管理
→ 允许自启动、允许关联启动、允许后台活动
```

**OPPO 手机：**
```
设置 → 电池 → 应用耗电管理 → 伴学 → 允许后台运行
设置 → 应用管理 → 伴学 → 权限 → 自启动 → 开启
```

**VIVO 手机：**
```
设置 → 电池 → 后台高耗电 → 伴学 → 允许
设置 → 更多设置 → 权限管理 → 自启动 → 伴学 → 开启
```

### API 配置 404 问题
如果遇到 404 错误，说明服务器上缺少 `backend/routers/api_config.py` 文件。
运行 `.\deploy_v1.2.0.ps1` 会自动上传并配置。

### 后台保活限制
- 无法 100% 保证不被杀
- 内存严重不足时系统仍会杀掉应用
- 用户手动清理后台会终止应用
- 省电模式下可能被限制
