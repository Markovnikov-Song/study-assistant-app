# 快速部署 v1.2.0

## 一键部署到服务器

```powershell
.\deploy_v1.2.0.ps1
```

输入服务器密码，等待完成。

## 构建新版本 APK

```powershell
flutter pub get
flutter clean
flutter build apk --release
```

## 上传 APK 到服务器

```powershell
scp build/app/outputs/flutter-apk/app-release.apk admin@47.104.165.105:/home/admin/study-assistant-app/backend/downloads/app-v1.2.0.apk
```
