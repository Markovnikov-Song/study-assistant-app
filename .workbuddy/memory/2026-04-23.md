## 数据库优化（2026-04-24 完成）

**问题分析：** 对 `backend/database.py` 和相关代码进行了详细分析，发现以下问题：

1. **HintSuggestion 表唯一约束缺失**：`hint_type` 未包含在索引中，且缺少 UNIQUE 约束
2. **多个表缺少索引**：`chunks`、`documents`、`past_exam_files`、`past_exam_questions`、`conversation_sessions`、`conversation_history`、`subjects`
3. **N+1 查询问题**：`routers/notes.py` 中 `batch_create_notes` 循环查询笔记本

**修复内容：**

1. `database.py`:
   - `HintSuggestion` 表：添加 `UniqueConstraint("user_id", "subject_id", "hint_type")`
   - `Chunk` 表：添加 `idx_chunks_subject_id` 索引
   - `Document` 表：添加 `idx_documents_user_id` 和 `idx_documents_subject_id` 索引
   - `PastExamFile` 表：添加 `idx_past_exam_files_user_id` 和 `idx_past_exam_files_subject_id` 索引
   - `PastExamQuestion` 表：添加 `idx_past_exam_questions_subject_id` 索引
   - `ConversationSession` 表：添加 `idx_conversation_sessions_user_id` 和 `idx_conversation_sessions_user_created` 索引
   - `ConversationHistory` 表：添加 `idx_conversation_history_session_id` 和 `idx_conversation_history_created_at` 索引
   - `Subject` 表：添加 `idx_subjects_user_id` 索引

2. `routers/notes.py`:
   - `batch_create_notes` 函数：将 N+1 循环查询改为批量 IN 查询

3. 新增 `migrations/008_add_missing_indexes.sql`：
   - 添加所有缺失索引的 SQL 语句
   - 包含 hint_suggestions 约束修复的 DO 块

**验证：** Python 语法检查通过，ORM 模型加载正常，无 lint 错误
